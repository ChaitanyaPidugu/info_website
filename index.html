Data Center Networks (with SIUE, USA June 2023 - July 2023)

During my tenure at SIUE, I undertook a comprehensive project focused on implementing overlay networks and microsegmentation within a virtualized environment. This project aimed to demonstrate effective communication through overlay and underlay networks while implementing robust security measures tailored to the specific roles of individual virtual machines (VMs).

Instrument Cluster Testing (with Progile Infotech, India Jan 2021 - Feb 2022)

This project involved validation of instrument panel cluster connected to a hardware test bench against the functional specifications. Validation of communication protocols like CANFD and LIN. Checking the functionality of Dataflash, verifying Bootloader functionality of cluster. Other features tested were HUD, Aging and Teltales. Verifying the power levels at different pins by measuring voltage, current values by using probes and CROs. Most importantly, performed diagnostics to the cluster using Unified Diagnostic Service protocols.

Digital Eye For Coach Guidance Automation (with South Central Railway Station Vijayawada, India Jan 2017 - April 2017)

Established a program to feed the coach numbers automatically before the train is entering the station reducing manual interference. Implemented RFID receivers, transmitters, and tags to wirelessly transfer data from passive tag. Tracked and identified the coach numbers by passive tags.

Real Time Weather Monitoring System (with Vector India Pvt Ltd, India July 2019 - Dec 2019)

Collected and stored various parameters like temperature, light intensity data using different sensors. Implemented a logger system which converts this data into a meaningful format using 8051 development board.
